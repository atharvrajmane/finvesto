# Finvesto 💹

**Finvesto** is a modern web application designed to provide a clean, responsive, and professional user interface for a financial investment platform. The project is built to practice full-stack development and frontend design principles.

---

## 🚀 Features

- ✅ Modern, responsive UI
- ✅ Built with React and Bootstrap
- ✅ Component-based architecture
- ✅ Clean folder structure
- ✅ Future scope: Authentication, dashboard, and integration with APIs

---

## 🛠️ Tech Stack

| Technology            | Purpose/Role                           |
|------------------------|----------------------------------------|
| **React JS**           | Frontend framework                     |
| **Axios**              | HTTP requests                          |
| **Vite**               | Frontend build tool                    |
| **Material UI (MUI)**  | Modern React UI components             |
| **Emotion**            | Styling library used with MUI          |
| **Bootstrap**          | Legacy styling                       |
| **Node.js**            | JavaScript runtime for backend         |
| **Express.js**         | Backend framework                      |
| **MongoDB**            | NoSQL database                         |
| **CORS**               | Cross-origin handling                  |
